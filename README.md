# Prerna Pawar Portfolio

A professional, problem-solver focused portfolio website built with Next.js, React, and Tailwind CSS.

## Features
- **Modern Tech Stack**: Next.js 14 (App Router), TypeScript, Tailwind CSS v4.
- **Animations**: Framer Motion for smooth, accessible interactions.
- **Dark Mode**: Polished dark-themed UI.
- **Content**: Tailored to Prerna Pawar's resume (Urja Foods, Innovexa).

## Getting Started

### Prerequisites
- Node.js (v18 or later)
- npm

### Installation

1.  **Install dependencies**:
    ```bash
    npm install
    ```

### Running Locally

1.  **Start the development server**:
    ```bash
    npm run dev
    ```

2.  **Open in browser**:
    Visit [http://localhost:3000](http://localhost:3000)

### Environment Variables (Optional)
This project uses MongoDB for dynamic data (optional for UI viewing).
Create a `.env.local` file in the root directory:
```env
MONGODB_URI=your_mongodb_connection_string
```
*Note: The app will run fine without this, using static fallback content.*

## Deployment
This application is ready for deployment on [Vercel](https://vercel.com).
1. Push to GitHub.
2. Import project in Vercel.
3. Add environment variables.
4. Deploy.
